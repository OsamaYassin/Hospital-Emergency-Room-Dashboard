# Hospital Emergency Room Dashboard

A Power BI data analytics project focused on Emergency Room (ER) performance monitoring, patient flow analysis, and operational decision support.

## Project Overview

This dashboard helps healthcare teams and administrators monitor ER activity, identify bottlenecks, and improve service quality using data-driven insights.

## Objectives

- Track patient volume, wait times, and treatment timelines
- Monitor department-level operational performance
- Analyze peak hours, admission patterns, and discharge outcomes
- Support staffing and resource planning decisions

## Key Metrics (KPIs)

- Total ER Visits
- Average Wait Time
- Average Time to Treatment
- Admission vs. Discharge Rate
- Left Without Being Seen (LWBS)
- Patient Throughput by Hour/Day

## Dashboard Scope

The Power BI report is designed to provide:

- **Executive Summary View** for high-level performance monitoring
- **Operational Analysis View** for queue, triage, and treatment flow
- **Trend Analysis View** for time-based demand and outcomes

## Data Sources

Typical inputs for this project may include:

- Hospital Information System (HIS) exports
- ER registration and triage data
- Patient encounter/treatment records
- Staffing and shift datasets
